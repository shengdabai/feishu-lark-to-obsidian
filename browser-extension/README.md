# Feishu/Lark to Obsidian Extension

这个插件的目标是：

- 打开飞书文档或知识库页面
- 点击一次浏览器扩展图标
- 自动把当前页面下载成 Markdown
- 同时下载一份原始纯文本备份
- 文件保存到系统下载目录下的 `Feishu Exports` 文件夹

## 安装

### Chrome / Edge

1. 打开扩展管理页：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. 打开右上角的“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择这个目录：

`browser-extension/`

5. 建议把插件固定到工具栏

## 使用

1. 在浏览器里打开一个你有查看权限的飞书页面
2. 等页面正文基本加载出来
3. 点击插件图标一次
4. 稍等几秒，插件会自动下载两个文件到 `Downloads/Feishu Exports/`

## 下载结果

- `文档标题.md`
- `文档标题.raw.txt`

其中：

- `.md` 适合直接放进 Obsidian
- `.raw.txt` 用来兜底保全文字，方便核查是否有遗漏

## 成功和失败提示

- 图标角标显示 `OK`：下载成功
- 图标角标显示 `ERR`：页面不对、权限不足，或者当前页没提取到正文

把鼠标悬停到插件图标上，可以看到更具体的状态说明。

## 注意

- 不能绕过飞书权限，前提仍然是你在页面里能看到内容
- 对飞书这种动态页面，建议先等正文加载出来再点
- 某些复杂卡片、嵌入块、多维表格不一定能完美转成 Markdown，但文字会尽量保存在 `.raw.txt` 里
